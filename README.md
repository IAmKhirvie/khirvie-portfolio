# Khirvie Clifford N. Bautista Portfolio

A static, GitHub Pages-ready portfolio for software developer, full-stack web developer, and IT specialist roles.

## Files

- `index.html` - Single-page portfolio markup
- `styles.css` - Premium dark UI, responsive layout, and animations
- `script.js` - Mobile navigation, active section highlighting, scroll reveals, and contact form behavior
- `resume.html` - Print-friendly ATS resume page linked from the portfolio

## Publish With GitHub Pages

1. Push this folder to a GitHub repository.
2. Open the repository settings.
3. Go to **Pages**.
4. Set the source to the `main` branch and root folder.
5. Save and wait for the Pages URL.

No build step is required.
